Step 1.
Open CMDwindow for Windows, and Bash for Linux.

Step 2.
Type command for 'make all' for both window.

Step 3.
For windows, type chat_server <port>
	     type chat_browser<ip> <port>
	     type chat_writer <ip><port><nickname>
For linux,   type ./chat_server <port>
	     type ./chat_browser<ip> <port>
	     type ./chat_writer <ip><port><nickname>

Step 4. 
Try for multi connections.

Step 5. 
If you want to delete all created files, type command 'make clean'.
